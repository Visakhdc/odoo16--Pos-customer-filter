POS CUSTOMER FILTER

This module helps to filter the customer's list in pos with their type.
And also it restrict pos sale for allowed users



Filtering Customer List

Step 1:

Contact -> Select a customer -> Select Customer Type.








Step 2:

Point of sale -> Select customer button -> Can be see Four buttons








Select Loyal Button







Select Impulse Button







Select Potential Button







Select All Button










POS RESTRICT CASHIER




Step 1:

Employees -> Select any employee -> Select Restrict Pos Sale.








Step 2:

Point Of Sale -> select cashier -> Can be see numpad is hide for 'Restrict Pos Sale' enabled users













Step 3:

Can be see numpad is visible for 'Restrict Pos Sale' disabled users
